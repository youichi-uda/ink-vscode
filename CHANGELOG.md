# Changelog

## 0.1.0 — 2026-03-15

Initial release.

### Features

- Syntax highlighting for `.ink` and `.ink2` files
- LSP-based language server with IntelliSense, Go to Definition, Find All References, Rename, and Hover
- Real-time diagnostics via inkjs compiler
- Interactive story preview with choice selection, variable watch, and knot jumping
- Branch visualizer (interactive story graph with dead code detection)
- Statistics panel (word count, playtime estimation, unused variable detection)
- Sidebar with Story Outline and Variables & Lists views
- Code folding for knots, stitches, conditional blocks, and comments
- 16 code snippets for common Ink patterns
- Compile to JSON
- Export localization to CSV
- Multi-file project support via INCLUDE
